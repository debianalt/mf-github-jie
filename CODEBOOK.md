## Decoupling State Codes

Following Tapio (2005), decoupling states are defined by the elasticity *e* = (%Delta MF) / (%Delta GDP):

| Code | Name | Definition |
|------|------|------------|
| SD | Strong Decoupling | *e* < 0, GDP growth > 0 |
| WD | Weak Decoupling | 0 <= *e* < 0.8, GDP growth > 0 |
| EC | Expansive Coupling | 0.8 <= *e* < 1.2, GDP growth > 0 |
| END | Expansive Negative Decoupling | *e* >= 1.2, GDP growth > 0 |
| RD | Recessive Decoupling | *e* < 0, GDP growth < 0 |
| RC | Recessive Coupling | 0.8 <= *e* < 1.2, GDP growth < 0 |
| WND | Weak Negative Decoupling | 0 <= *e* < 0.8, GDP growth < 0 |
| SND | Strong Negative Decoupling | *e* >= 1.2, GDP growth < 0 |
| NA | Not Available | Missing data |

## Cluster Definitions

Clusters are based on **temporal decoupling trajectory patterns** (sequences of decoupling states over 2000-2024), **not** on absolute income, GDP, or development levels. Countries with vastly different economic conditions can exhibit similar temporal patterns in their decoupling dynamics.

| Cluster | Label | n | Countries | Key characteristics |
|---------|-------|---|-----------|---------------------|
| 1 | Progressive Trajectories | 8 | BR, CY, DK, IE, NL, PY, PL, SE | Highest SD persistence (0.34), strongest upgrading (WD->SD = 0.43) |
| 2 | Stable Performers | 7 | HR, FR, GR, MT, PT, ES, UY | Lowest recoupling risk (0.13), lowest coupling persistence (0.10) |
| 3 | Atypical Trajectories | 3 | AR, EE, FI | Lowest SD persistence (0.09), extreme state jumps |
| 4 | Broad Convergence | 13 | AT, BE, BG, CZ, DE, HU, IT, LV, LT, LU, RO, SK, SI | Moderate dynamics, lowest transition entropy (12.5 bits) |

**Note:** Brazil and Paraguay (MERCOSUR) clustering with Denmark and Sweden (Cluster 1) reflects similar temporal transition patterns (SD/WD sequences), not economic similarity. Uruguay clustering with France and Spain (Cluster 2) reflects shared stability in decoupling trajectories. Argentina clustering with Estonia and Finland (Cluster 3) reflects shared trajectory volatility from macroeconomic instability.
